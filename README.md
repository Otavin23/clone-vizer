You must have the tools installed: <a href="https://git-scm.com/" target="_blank">Git</a>, <a href="https://nodejs.org/en" target="_blank">Node.js.</a>

<pre>
   # In the terminal, clone the project:

   # Enter in folder

   # Install dependecias
   $ npm install

   # Starting Project 
   $ npm run dev
